# flutter_prueba_backend

NestJS scaffold with `users` and `movimientos` modules using MongoDB via environment variables.

Quick start:

1. Copy `.env.example` to `.env` and set `MONGODB_URI`.
2. Run `npm install`.
3. Start in dev mode: `npm run start:dev`.

Endpoints:
- `GET /users`
- `POST /users`
- `GET /movimientos`
- `POST /movimientos`
